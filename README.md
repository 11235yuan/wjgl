# wjgl
此项目为比较简单的文件管理。主要包含三块
1.文件列表，能上传和下载文件
2.人员列表，能添加用户和给用户赋相应的角色，预定义了三种角色，普通用户、管理员、监察员
3.日志审计，方便监察员查看所有用户的上传和下载记录

后端采用python3.6 + django2.2
前端采用html + css + js

这是本人的第二个开源项目，希望你们能喜欢，有bug可以联系我(yangbhust@163.com)
